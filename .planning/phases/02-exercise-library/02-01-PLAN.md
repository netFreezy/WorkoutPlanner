**Create `Data/SeedData/ExerciseSeedData.cs`:**

A static class with two public static methods:
- `public static StrengthExercise[] GetStrengthExercises()` returning 37 strength exercises
- `public static EnduranceExercise[] GetEnduranceExercises()` returning 13 endurance exercises

All exercises use a fixed CreatedDate: `new DateTime(2026, 1, 1, 0, 0, 0, DateTimeKind.Utc)` -- never DateTime.UtcNow (causes spurious migration diffs per RESEARCH.md Pitfall 2).

ID ranges: Strength exercises use IDs 1-37, Endurance exercises use IDs 101-113. This leaves room for expansion and ensures user-created exercises (auto-increment from max ID + 1) start well above seed range.

**Strength exercise composition (per D-14, D-15):**
Calisthenics-focused with home equipment bias. Equipment distribution: primarily Bodyweight, Dumbbell, Kettlebell. Minimal Barbell/Machine. Use Equipment.Other for dip-bar specific exercises or just Bodyweight (since dip bars are a bodyweight station per RESEARCH.md Open Question 3).

Include exercises like:
- Back: Pull-Up, Chin-Up, Weighted Pull-Up, Inverted Row, Dumbbell Row, Dumbbell Pullover
- Chest: Push-Up, Weighted Dip, Diamond Push-Up, Dumbbell Bench Press, Dumbbell Fly, Incline Push-Up (feet elevated)
- Shoulders: Pike Push-Up, Dumbbell Overhead Press, Lateral Raise, Dumbbell Front Raise, Face Pull (Band)
- Legs: Bodyweight Squat, Bulgarian Split Squat, Pistol Squat, Goblet Squat, Romanian Deadlift (Dumbbell), Lunge, Calf Raise, Step-Up
- Arms: Dumbbell Curl, Hammer Curl, Tricep Dip, Overhead Tricep Extension, Concentration Curl
- Core: Plank, Hanging Leg Raise, Ab Wheel Rollout, Russian Twist, Dead Bug, L-Sit
- FullBody: Burpee, Turkish Get-Up, Kettlebell Swing

Every exercise MUST have a non-empty Description string with brief form cues (per D-17). Example: `"Hang from bar with overhand grip shoulder-width apart. Pull until chin clears bar. Control the descent."`.

**Endurance exercise composition (per D-16):**
Mostly running and cycling variants:
- Run: Easy Run, Tempo Run, Interval Run, Long Run, Fartlek Run, Hill Sprints, Recovery Run
- Cycle: Easy Ride, Tempo Ride, Interval Ride, Long Ride, Hill Climb Ride
- Walk: Brisk Walk

Each endurance exercise MUST have a Description with guidance. Example for Easy Run: `"Conversational pace. You should be able to hold a full conversation. Keep heart rate in Zone 2."`.
