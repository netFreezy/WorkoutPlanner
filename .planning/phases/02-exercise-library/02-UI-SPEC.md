# Phase 2 — UI Design Contract

> Visual and interaction contract for the exercise library phase. Elevated design targeting premium fitness app aesthetic. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (plain CSS + Blazor components) |
| Icon library | none (SVG inline icons for +, x, search, chevron; Unicode fallback prohibited) |
| Font | Inter (`'Inter', -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`) loaded via Google Fonts `<link>` in App.razor |

**Source:** RESEARCH.md confirms no Bootstrap, no third-party component libraries. Project constraint: Blazor components only, no JavaScript frameworks. Inter chosen for its superior readability at small sizes, variable weight support, and modern geometric aesthetic that suits a fitness app. Load weights 400 and 600 only to minimize payload.

**Google Fonts link (add to App.razor `<head>`):**
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600&display=swap" rel="stylesheet">
```

## Color

### Theme: Dark mode, premium fitness aesthetic

The dark theme uses a near-black base with cool blue-gray undertones, electric indigo-violet accent, and category-specific glow colors for strength (blue) and endurance (emerald). This creates visual depth and a premium feel suitable for a fitness app.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0A0A0F` | Page background, primary surface |
| Secondary (30%) | `#12121A` / `#1A1A25` | Card backgrounds, nav bar, filter bar, dialog panel |
| Accent (10%) | `#7C5CFC` | FAB, primary submit button, active nav indicator, focus rings, gradient accents |
| Destructive | `#EF4444` | Not used this phase (no delete functionality) |

### Semantic Tag Colors

| Tag | Background | Text | Border | Glow (hover) |
|-----|------------|------|--------|--------------|
| Strength | `rgba(59, 130, 246, 0.12)` | `#60A5FA` | `rgba(59, 130, 246, 0.25)` | `0 0 12px rgba(59, 130, 246, 0.15)` |
| Endurance | `rgba(16, 185, 129, 0.12)` | `#34D399` | `rgba(16, 185, 129, 0.25)` | `0 0 12px rgba(16, 185, 129, 0.15)` |

**WCAG contrast verification (dark theme):**
- Primary text `#F0F0F5` on `#0A0A0F` = 18.2:1 (AAA)
- Primary text `#F0F0F5` on `#1A1A25` = 14.1:1 (AAA)
- Secondary text `rgba(240,240,245,0.60)` on `#0A0A0F` = ~8.5:1 (AAA)
- Strength tag `#60A5FA` on `rgba(59,130,246,0.12)` over `#1A1A25` = ~7.8:1 (AA)
- Endurance tag `#34D399` on `rgba(16,185,129,0.12)` over `#1A1A25` = ~8.1:1 (AA)
- Accent `#7C5CFC` on `#0A0A0F` = 5.1:1 (AA for normal text, AAA for large)

## Accessibility Contract

| Concern | Implementation |
|---------|---------------|
| Dark theme contrast | All text meets WCAG 2.1 AA minimum (see Color section for specific ratios). Accent on dark backgrounds verified at 5.1:1. |
| Dialog focus trap | On dialog open, focus moves to first focusable element. Tab cycles within dialog only. Focus returns to trigger element on close. |
| Dialog close on Escape | Pressing Escape key closes any open dialog |
| Dialog close button | `aria-label="Close dialog"` on SVG x button |
| Dialog role | `role="dialog"` and `aria-modal="true"` on dialog panel, `aria-labelledby` pointing to title element |
| Card keyboard access | Cards are focusable (`tabindex="0"`) and activate on Enter/Space. Focus ring: `2px solid var(--color-accent), 2px offset` |
| Filter labels | Each select has a visible `<label>` element. If space-constrained, use `aria-label` matching the default option text |
| Filter chip remove | `aria-label="Remove {filter type} filter"` on each chip's x button |
| Search input | `<label>` element (visually hidden via `sr-only` class) with text "Search exercises" |
| FAB | `aria-label="Add Exercise"` on button |
| Toast | `role="status"` with `aria-live="polite"` |
| Color not sole indicator | Type tags use text labels ("Strength"/"Endurance") in addition to color |
| Focus visible | All interactive elements show visible focus indicator: `outline: 2px solid var(--color-accent); outline-offset: 2px` |
| Reduced motion | All animations wrapped in `@media (prefers-reduced-motion: no-preference)`. With reduced motion preference, transitions resolve at 0ms, no keyframe animations play |
| Scrollbar styling | Custom thin scrollbar for dialogs: `scrollbar-width: thin; scrollbar-color: var(--color-border-subtle) transparent` |
