Insert this markup before the `<div class="detail-footer">` line:

```razor
@if (Workout.Status == WorkoutStatus.Planned && Workout.WorkoutTemplateId != null)
{
    <div class="detail-divider"></div>
    <button class="btn-start-session" @onclick="HandleStartSession">
        <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">
            <polygon points="5 3 19 12 5 21 5 3"/>
        </svg>
        Start Session
    </button>
}
```

3. Add the handler method in the `@code` block:
```csharp
private void HandleStartSession()
{
    if (Workout == null) return;
    NavigationManager.NavigateTo($"/session/{Workout.Id}");
}
```
