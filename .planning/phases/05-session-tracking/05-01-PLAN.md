using BlazorApp2.Data;
using BlazorApp2.Data.Entities;

**Methods to implement (each creates and disposes its own DbContext via `await using var context = await _contextFactory.CreateDbContextAsync()`):**

a) `public async Task<WorkoutLog> StartSessionAsync(int scheduledWorkoutId)`:
   - Load ScheduledWorkout with `.Include(sw => sw.WorkoutTemplate!).ThenInclude(t => t.Items.OrderBy(i => i.Position)).ThenInclude(i => i.Exercise)` and `.Include(sw => sw.WorkoutLog)`
   - If `scheduled.WorkoutLog != null`, call and return `LoadSessionAsync(scheduled.WorkoutLog.Id)` (resume case per D-15)
   - Create new `WorkoutLog { ScheduledWorkoutId = scheduledWorkoutId, StartedAt = DateTime.UtcNow }`
   - For each TemplateItem ordered by Position:
     - If `item.Exercise is StrengthExercise`: create `item.TargetSets ?? 1` SetLog rows with `SetType = SetType.Working`, `PlannedReps = item.TargetReps`, `PlannedWeight = item.TargetWeight`, `ActualReps = item.TargetReps`, `ActualWeight = item.TargetWeight`, `IsCompleted = false` (pre-fill actual with planned per D-01)
     - If `item.Exercise is EnduranceExercise endEx`: create one EnduranceLog with `ActivityType = endEx.ActivityType`, all Planned fields from template targets, pre-fill `ActualDistance` and `ActualDurationSeconds` from planned, `IsCompleted = false`
   - Set `scheduled.Status = WorkoutStatus.Completed` (mark as started)
   - Save and return the WorkoutLog via `LoadSessionAsync`

b) `public async Task<WorkoutLog> LoadSessionAsync(int workoutLogId)`:
   - Load WorkoutLog with `.Include(wl => wl.SetLogs.OrderBy(sl => sl.SetNumber)).ThenInclude(sl => sl.Exercise)`, `.Include(wl => wl.EnduranceLogs).ThenInclude(el => el.Exercise)`, `.Include(wl => wl.ScheduledWorkout).ThenInclude(sw => sw.WorkoutTemplate)`
   - Return the loaded entity (throw InvalidOperationException if not found)

c) `public async Task CompleteSetAsync(int setLogId, double? actualWeight, int? actualReps, SetType setType)`:
   - Find SetLog by ID, update `ActualWeight`, `ActualReps`, `SetType`, set `IsCompleted = true`, save (per D-13)

d) `public async Task UncompleteSetAsync(int setLogId)`:
   - Find SetLog by ID, set `IsCompleted = false`, save

e) `public async Task SaveEnduranceLogAsync(int enduranceLogId, double? actualDistance, int? actualDurationSeconds, int? actualHeartRateZone)`:
   - Find EnduranceLog by ID, update fields, auto-calculate pace: `actualDistance > 0 && actualDurationSeconds > 0 ? (actualDurationSeconds.Value / 60.0) / actualDistance.Value : null` (min/km), set `ActualPace`, `IsCompleted = true`, save

f) `public async Task<SetLog> AddSetAsync(int workoutLogId, int exerciseId, double? prefillWeight)`:
   - Query max SetNumber for this exercise in this workoutLog, create new SetLog with SetNumber = max+1, SetType = Working, ActualWeight = prefillWeight, IsCompleted = false, save and return (per D-03)

g) `public async Task<List<PreviousStrengthSession>> GetPreviousStrengthPerformanceAsync(int exerciseId, int? excludeWorkoutLogId = null, int limit = 3)`:
   - Create a DTO record: `public record PreviousStrengthSession(DateTime Date, List<PreviousSet> Sets)`
   - Create a DTO record: `public record PreviousSet(int SetNumber, double? Weight, int? Reps)`
   - Query SetLogs where ExerciseId matches, IsCompleted = true, WorkoutLog.CompletedAt != null, exclude current workoutLogId if provided
   - Group by WorkoutLogId, order by WorkoutLog.StartedAt descending, take `limit`
   - Map to DTOs

h) `public async Task<List<PreviousEnduranceSession>> GetPreviousEndurancePerformanceAsync(int exerciseId, int? excludeWorkoutLogId = null, int limit = 3)`:
   - Create a DTO record: `public record PreviousEnduranceSession(DateTime Date, double? Distance, int? DurationSeconds, double? Pace)`
   - Query EnduranceLogs where ExerciseId matches, IsCompleted = true, WorkoutLog.CompletedAt != null, exclude current workoutLogId
   - Order by WorkoutLog.StartedAt descending, take `limit`
   - Map to DTOs

i) `public async Task FinishSessionAsync(int workoutLogId, int? rpe, string? notes)`:
   - Load WorkoutLog with ScheduledWorkout, set `CompletedAt = DateTime.UtcNow`, `Rpe = rpe`, `Notes = notes`, ensure `ScheduledWorkout.Status = WorkoutStatus.Completed`, save (per D-11)

j) `public async Task AbandonSessionAsync(int workoutLogId)`:
   - Load WorkoutLog with ScheduledWorkout, set `CompletedAt = DateTime.UtcNow`, ensure `ScheduledWorkout.Status = WorkoutStatus.Completed` (keeps partial data per D-16), save

k) `public async Task<int?> GetIncompleteSessionIdAsync()`:
   - Query WorkoutLogs where `CompletedAt == null`, select `ScheduledWorkoutId`, return first or null (per D-15)

l) `public async Task<List<ScheduledWorkout>> GetTodaysWorkoutsAsync()`:
   - Get today's date (UTC), query ScheduledWorkouts where `ScheduledDate.Date == today` and `Status == WorkoutStatus.Planned`, include WorkoutTemplate and WorkoutLog, order by ScheduledDate, return list (per D-09)

m) `public async Task UpdateSetTypeAsync(int setLogId, SetType setType)`:
   - Find SetLog, update SetType, save (per D-02, persist type change immediately)

Place the DTO records at the bottom of the file, outside the class but inside the namespace.
