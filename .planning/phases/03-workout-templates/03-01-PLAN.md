     - `BuilderGroup` with properties: `string LocalId` (Guid.NewGuid().ToString()), `GroupType GroupType`, `int? Rounds`, `int? MinuteWindow`
     - `TemplateSnapshot` (private record): `string Json` (serialized state for undo/redo)
     - `TemplateBuilderState` with:
       - Properties: `string Name`, `string? Description`, `List<string> Tags = new()`, `List<BuilderItem> Items = new()`, `List<BuilderGroup> Groups = new()`, `bool HasUnsavedChanges`, `bool CanUndo => _undoStack.Count > 0`, `bool CanRedo => _redoStack.Count > 0`
       - Private fields: `Stack<string> _undoStack = new()`, `Stack<string> _redoStack = new()`, max const `MaxUndoDepth = 50`
       - `PushUndo()`: Serialize current state (Items + Groups + Name + Description + Tags) to JSON, push to _undoStack, clear _redoStack, set HasUnsavedChanges = true. If _undoStack.Count > MaxUndoDepth, trim oldest.
       - `Undo()`: If CanUndo, serialize current to JSON and push to _redoStack. Pop _undoStack, deserialize and restore Items/Groups/Name/Description/Tags.
       - `Redo()`: If CanRedo, serialize current to JSON and push to _undoStack. Pop _redoStack, deserialize and restore.
       - `ResetChangeTracking()`: HasUnsavedChanges = false, clear both stacks.
       - Static `EstimateDurationMinutes(List<BuilderItem> items, List<BuilderGroup> groups)`: returns int.
         - For EMOM groups: sum `(rounds ?? 1) * (minuteWindow ?? 1)` for each group. Track item LocalIds handled by EMOM.
         - For remaining strength items: `(targetSets ?? 2) * 1.5`
         - For remaining endurance items: `targetDurationSeconds.HasValue ? targetDurationSeconds.Value / 60.0 : 10`
         - Round result to nearest 5: `(int)(Math.Round(total / 5.0) * 5)`
         - Minimum return value: 5 (no "~0 min" displayed)
       - Private snapshot serialization: Create an anonymous object `{ Name, Description, Tags, Items = Items.Select(i => new { ... all props except IsSelected }), Groups }` and serialize to JSON. Deserialization restores all fields.
