## Visual Hierarchy

**Template list screen focal points:**
- **Primary focal point:** The FAB ("Create Template") in the bottom-right corner. Its accent gradient, 56px size, and elevation shadow make it the highest-contrast element on the page and the first call to action for new users.
- **Secondary focal point:** The first template card in the grid (top-left position). For returning users with existing templates, the top-left card draws the eye as the natural reading start point. The staggered `fadeSlideIn` animation reinforces this by revealing the top-left card first.
- **Empty state focal point:** When no templates exist, the centered empty state illustration and "Create Template" ghost button become the sole focal point, directing new users to their first action.

### Template Builder
| Interaction | Behavior |
|-------------|----------|
| Click "Add exercises" | Open ExercisePickerDialog (D-08) |

| Edit target field | Inline update. No blur-to-save; values are held in memory until explicit Save (D-10) |

| Click "Discard" | If unsaved changes exist, show DiscardConfirmationDialog. If confirmed or no changes, navigate to `/templates` (D-10) |
| Click "Undo" / Ctrl+Z | Undo last action in builder (D-11) |
| Click "Redo" / Ctrl+Shift+Z | Redo last undone action (D-11) |
| Click remove (X) on exercise | Remove exercise from template. If section becomes empty (warm-up or cool-down), remove section header (D-21) |
| Enter text in tag input + Enter | Add tag pill to template (D-02) |
| Backspace on empty tag input | Remove last tag pill |
| Browser back / nav away with unsaved changes | Show DiscardConfirmationDialog |

## Estimated Duration Calculation Heuristic

Display on template cards and detail view as "~N min".

Formula:
- Each strength exercise: `targetSets * 1.5 min` (includes rest between sets). Default 3 min if no sets specified.
- Each endurance exercise: `targetDurationSeconds / 60`. Default 10 min if no duration specified.
- Warm-up section: Add flat 5 min if present and no exercises in it have targets.
- Cool-down section: Add flat 5 min if present and no exercises in it have targets.
- EMOM groups: `rounds * minuteWindow` (replaces per-exercise calc for grouped exercises).
- Round to nearest 5 min for display.
