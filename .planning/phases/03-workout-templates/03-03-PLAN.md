2. **Create `Components/Pages/TemplateBuilder.razor.cs`** (code-behind):
   - Inject: `IDbContextFactory<AppDbContext> DbFactory`, `NavigationManager NavigationManager`.
   - `[Parameter] public int? Id { get; set; }`
   - Private fields: `TemplateBuilderState State = new()`, `bool isEditing`, `int? templateId`, `bool showExercisePicker`, `bool showDiscardDialog`, `bool showDescription`, `string? pendingNavigationUri`, `Toast toast = null!`.
   - `OnInitializedAsync`: if `Id` has value, load template from DB. Convert TemplateItems to BuilderItems and TemplateGroups to BuilderGroups. Set `isEditing = true`, `templateId = Id`. Set `showDescription = !string.IsNullOrEmpty(State.Description)`. Call `State.ResetChangeTracking()`.
   - Loading from DB: use `DbFactory.CreateDbContext()`, query `WorkoutTemplates.Include(t => t.Items).ThenInclude(ti => ti.Exercise).Include(t => t.Groups).ThenInclude(g => g.Items).FirstOrDefaultAsync(t => t.Id == Id)`. If not found, navigate to `/templates`.
   - Mapping entity to builder state:
     ```csharp
     State.Name = template.Name;
     State.Description = template.Description;
     State.Tags = new List<string>(template.Tags);
     // Map groups first so items can reference them
     var groupMap = new Dictionary<int, string>(); // entity ID -> local ID
     foreach (var g in template.Groups)
     {
         var bg = new BuilderGroup { GroupType = g.GroupType, Rounds = g.Rounds, MinuteWindow = g.MinuteWindow };
         State.Groups.Add(bg);
         groupMap[g.Id] = bg.LocalId;
     }
     foreach (var item in template.Items.OrderBy(i => i.Position))
     {
         State.Items.Add(new BuilderItem
         {
             ExerciseId = item.ExerciseId,
             ExerciseName = item.Exercise.Name,
             IsStrength = item.Exercise is StrengthExercise,
             Position = item.Position,
             SectionType = item.SectionType,
             GroupLocalId = item.TemplateGroupId.HasValue && groupMap.ContainsKey(item.TemplateGroupId.Value) ? groupMap[item.TemplateGroupId.Value] : null,
             TargetSets = item.TargetSets, TargetReps = item.TargetReps, TargetWeight = item.TargetWeight,
             TargetDistance = item.TargetDistance, TargetDurationSeconds = item.TargetDurationSeconds,
             TargetPace = item.TargetPace, TargetHeartRateZone = item.TargetHeartRateZone
         });
     }
     ```
   - `OnNameChanged()`: `State.PushUndo();` (only if name actually changed from last snapshot -- track a `_lastSavedName` field).
   - `OnTagsChanged(List<string> tags)`: `State.PushUndo(); State.Tags = tags;`
   - `OnTargetChanged(BuilderItem item)`: `State.PushUndo();` (the item is already mutated by the ExerciseRow).
   - `OnSelectionChanged()`: `StateHasChanged();` (no undo for selection toggle).
   - `AddSelectedExercises(List<Exercise> exercises)`: per D-08, D-19:
     ```csharp
     State.PushUndo();
     int nextPosition = State.Items.Count > 0 ? State.Items.Max(i => i.Position) + 1 : 0;
     foreach (var ex in exercises)
     {
         State.Items.Add(new BuilderItem
         {
             ExerciseId = ex.Id,
             ExerciseName = ex.Name,
             IsStrength = ex is StrengthExercise,
             Position = nextPosition++,
             SectionType = SectionType.Working  // D-19: default to Working
         });
     }
     showExercisePicker = false;
     ```
   - `RemoveExercise(BuilderItem item)`:
     ```csharp
     State.PushUndo();
     State.Items.Remove(item);
     // Recompact positions
     for (int i = 0; i < State.Items.Count; i++)
         State.Items[i].Position = i;
     ```
   - `SaveTemplate()`:
     - If `string.IsNullOrWhiteSpace(State.Name)`, show toast "Template name is required" and return.
     - Use fresh DbContext. Follow RESEARCH save pattern:
       - If editing: load existing with includes, RemoveRange items and groups, update Name/Description/Tags.
       - If new: create new WorkoutTemplate.
     - Rebuild groups with groupMap (BuilderGroup.LocalId -> TemplateGroup entity).
     - Rebuild items with ExerciseId, Position, SectionType, target fields, group mapping.
     - SaveChangesAsync.
     - `State.ResetChangeTracking()`. If was new, navigate to `/templates/{newId}/edit` so user is now in edit mode.
     - Show toast "Template saved".
   - `Discard()`: if `State.HasUnsavedChanges`, show discard dialog. Else navigate to `/templates`.
   - `OnBeforeInternalNavigation(LocationChangingContext context)`: if `State.HasUnsavedChanges`, prevent navigation, store `pendingNavigationUri = context.Uri`, show discard dialog.
   - `ConfirmDiscard()`: navigate to `pendingNavigationUri ?? "/templates"`.
   - `CancelDiscard()`: `showDiscardDialog = false; pendingNavigationUri = null;`
   - `Undo()`: `State.Undo(); StateHasChanged();`
   - `Redo()`: `State.Redo(); StateHasChanged();`
   - `OpenExercisePicker()`: `showExercisePicker = true;`
   - `CloseExercisePicker()`: `showExercisePicker = false;`
   - Properties for toolbar: `SelectedCount => State.Items.Count(i => i.IsSelected)`, `HasSelection => SelectedCount > 0`.
