## Visual Hierarchy

**Calendar page focal points:**
- **Primary focal point:** The current week grid. Seven columns spanning the full content width, with today's column subtly highlighted via `--color-accent-muted` background. The today date number rendered in a filled accent circle provides the strongest color contrast on the page.
- **Secondary focal point:** The FAB ("Schedule Workout") in the bottom-right corner. Accent gradient, 56px, consistent with Exercises and Templates pages.
- **Tertiary focal point:** The monthly mini-calendar toggle panel (when open). Color-coded dots provide a density overview that draws the eye for planning context.
- **Empty state focal point:** When no workouts are scheduled for the visible week, centered empty state with "Plan your week" heading and a ghost button to schedule the first workout.

### Calendar Page

| Interaction | Behavior |
|-------------|----------|
| Click prev arrow | Navigate to previous week (weekly) or month (monthly). Grid animates `weekSlideRight` (content slides from left). |
| Click next arrow | Navigate to next week (weekly) or month (monthly). Grid animates `weekSlideLeft` (content slides from right). |
| Click "Today" | Jump to current week (weekly) or current month (monthly). No animation -- instant jump. |
| Click "Week" toggle | Switch to weekly view. Load data for displayed week. |
| Click "Month" toggle | Switch to monthly view. Load monthly dot data. |
| Click empty day cell | Open ScheduleDialog with clicked date pre-filled (D-05). |
| Click "+" in day cell | Open ScheduleDialog with clicked date pre-filled (D-05). |
| Click FAB | Open ScheduleDialog with today's date pre-filled. |
| Click workout chip | Open WorkoutDetailDialog for that workout (D-07). |
| Drag workout chip to another day | Move scheduled workout to new date (D-08). Show drag placeholder in target cell. Toast "Workout moved to {day name}". If recurring, only move this single occurrence (detach from recurrence). |
| Click monthly date cell | Switch to weekly view for that date's week (D-02). |

### Schedule Dialog

| Interaction | Behavior |
|-------------|----------|
| Toggle "From Template" / "Ad-hoc" | Switch between template picker and ad-hoc name input. |
| Select template in picker | Highlight selected template. Only one can be selected at a time. |
| Search templates | Filter template list client-side by name. |
| Toggle "Repeat" on | Reveal recurrence options with `sectionExpand` animation. |
| Toggle "Repeat" off | Hide recurrence options. Clear recurrence state. |
| Select "Weekly" frequency | Show day-of-week toggle chips. Pre-select the current day of the scheduled date. |
| Select "Every X days" frequency | Show interval number input. Default value: 2. |
| Click day-of-week chip | Toggle that day on/off (multi-select, maps to DaysOfWeek flags). At least one day must be selected (prevent deselect of last active day). |
| Click "Schedule" | Validate (template or name required, date required). Save ScheduledWorkout. If recurring, save RecurrenceRule and trigger materialization. Toast confirmation. Close dialog. Refresh calendar. |
| Click "Cancel" | Close dialog, no changes. |
