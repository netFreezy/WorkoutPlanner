    5. **Create Services directory** and add `MaterializationService.cs`:
       - Constructor takes `IDbContextFactory<AppDbContext>`
       - Add static helper `ToDaysOfWeek(System.DayOfWeek)` mapping: Sunday->DaysOfWeek.Sunday, Monday->DaysOfWeek.Monday, etc. (explicit switch, not cast -- different ordinal values)
       - Add static helper `GetMondayOfWeek(DateTime)`: `date.Date.AddDays(-((7 + (int)date.DayOfWeek - (int)System.DayOfWeek.Monday) % 7))`
       - Add `GenerateOccurrences(RecurrenceRule rule, DateTime windowStart, DateTime windowEnd)` returning `IEnumerable<DateTime>`:
         - FrequencyType.Daily: Start from rule.StartDate, iterate by rule.Interval days. Only yield dates within [windowStart, windowEnd).
         - FrequencyType.Weekly: Iterate weeks from GetMondayOfWeek(windowStart) to windowEnd. For each week, check 7 days. Yield those where ToDaysOfWeek(day.DayOfWeek) is flagged in rule.DaysOfWeek and day is in [windowStart, windowEnd).
         - FrequencyType.Custom: Same as Weekly but step by 7*rule.Interval days between weeks.
       - Add `MaterializeAsync(int recurrenceRuleId)`:
         - Load rule with Include(r => r.ScheduledWorkouts)
         - windowStart = DateTime.UtcNow.Date, windowEnd = windowStart.AddDays(28)
         - Get existing dates as HashSet<DateTime>
         - For each generated date not in set, create new ScheduledWorkout with: ScheduledDate=date, WorkoutTemplateId=rule.WorkoutTemplateId, AdHocName=rule.AdHocName, RecurrenceRuleId=rule.Id, Status=Planned
         - SaveChangesAsync
       - Add `MaterializeAllAsync()`: Load all rule IDs, call MaterializeAsync for each

    6. **Create `Services/SchedulingService.cs`**:
       - Constructor takes `IDbContextFactory<AppDbContext>` and `MaterializationService`
       - `GetWorkoutsForWeekAsync(DateTime weekStart)`: Query ScheduledWorkouts where ScheduledDate >= weekStart and < weekStart.AddDays(7), Include WorkoutTemplate.Items.Exercise and RecurrenceRule, OrderBy ScheduledDate. Return as List.
       - `GetWorkoutsForMonthAsync(DateTime monthStart)`: Query where ScheduledDate >= monthStart and < monthStart.AddMonths(1), same includes. Return as List.
       - `ScheduleWorkoutAsync(int? templateId, string? adHocName, DateTime date, RecurrenceRule? recurrence)`: Create ScheduledWorkout, optionally create RecurrenceRule and call MaterializeAsync. Return the created workout.
       - `RemoveWorkoutAsync(int workoutId)`: Delete single workout. If it was the last occurrence of a recurrence, delete the rule too.
       - `RemoveRecurringAsync(int recurrenceRuleId)`: Delete rule and all future Planned occurrences (not Completed/Skipped).
       - `SkipWorkoutAsync(int workoutId)`: Set Status = Skipped.
       - `RescheduleWorkoutAsync(int workoutId, DateTime newDate)`: Update ScheduledDate. If recurring, detach from recurrence (set RecurrenceRuleId = null).
       - Add static `DetermineWorkoutType(ScheduledWorkout sw)` returning a WorkoutType enum (define in same file or a new enum file): returns AdHoc if template is null, Strength if all items are StrengthExercise, Endurance if all EnduranceExercise, Mixed if both.
