State fields:
```csharp
private List<HistorySession> sessions = new();
private List<(int Id, string Name)> loggedExercises = new();
private HashSet<int> expandedCards = new();
private string searchText = "";
private bool filterStrength = false;
private bool filterEndurance = false;
private bool filterMixed = false;
private DateTime? dateStart;
private DateTime? dateEnd;
private int? selectedExerciseId;
private int totalCount = 0;
private int loadedCount = 0;
private bool isLoading = true;
private bool isLoadingMore = false;
private const int PageSize = 20;
private System.Timers.Timer? _debounceTimer;
```

OnInitializedAsync: Load first 20 sessions via `GetCompletedSessionsAsync(null, null, null, 0, PageSize)`. Load logged exercises via `GetLoggedExercisesAsync()`. Get total count via `GetTotalCountAsync(null, null, null)`. Set loadedCount = sessions.Count, isLoading = false.

Methods:
- `private async Task LoadMore()`: isLoadingMore = true, fetch next page `GetCompletedSessionsAsync(dateStart, dateEnd, selectedExerciseId, loadedCount, PageSize)`, append to sessions, update loadedCount and totalCount, isLoadingMore = false.
- `private async Task ApplyFilters()`: Reset sessions, loadedCount = 0, call `GetCompletedSessionsAsync(dateStart, dateEnd, selectedExerciseId, 0, PageSize)`, update totalCount. Client-side filter by type chips (strength/endurance/mixed) and search text is applied in the Razor template since type and search are client-side filters (per D-13: text search is case-insensitive contains on workout/exercise name, debounced 300ms).
- `private void OnSearchInput(ChangeEventArgs e)`: Update searchText, use debounce timer (300ms) before triggering StateHasChanged (client-side filter).
- `private void ToggleStrength()`, `ToggleEndurance()`, `ToggleMixed()`: Toggle respective bools, client-side filter.
- `private async Task OnDateRangeChanged()`: Re-query DB with dateStart/dateEnd (per interaction contract: date range change re-queries DB).
- `private async Task OnExerciseFilterChanged(ChangeEventArgs e)`: Set selectedExerciseId, re-query DB.
- `private void ToggleCard(int workoutLogId)`: Add/remove from expandedCards HashSet.
- `private bool IsExpanded(int workoutLogId)`: expandedCards.Contains(workoutLogId).
- `private IEnumerable<HistorySession> GetFilteredSessions()`: Apply client-side filters -- search text (contains on WorkoutName or any ExerciseName), type chips (if any active, filter to matching WorkoutType). Return filtered list.
- `private int GetRemainingCount()`: totalCount - loadedCount (clamped to 0).
- `private async Task ClearFilters()`: Reset all filters, re-query from scratch.

Dispose: clean up _debounceTimer.

**History.razor** -- Page markup:

```razor
@page "/history"
@rendermode InteractiveServer
@implements IDisposable

<PageTitle>History</PageTitle>

<div class="history-page">
    <h1 class="page-title">History</h1>

    @if (isLoading)
    {
        <!-- Loading skeleton -->
    }
    else
    {
        <!-- Filter Bar -->
        <div class="filter-bar">
            <!-- Search input with search icon -->
            <div class="search-wrapper">
                <svg class="search-icon"><!-- 16x16 search/magnifying glass SVG --></svg>
                <input type="search" class="search-input" placeholder="Search workouts or exercises..."
                       aria-label="Search workout history"
                       value="@searchText" @oninput="OnSearchInput" />
            </div>

            <!-- Filter row: type chips + date range + exercise filter -->
            <div class="filter-row">
                <div class="type-chips" role="group" aria-label="Filter by workout type">
                    <button class="filter-chip @(filterStrength ? "filter-chip--active" : "")"
                            aria-pressed="@filterStrength" @onclick="ToggleStrength">
                        <span class="chip-dot chip-dot--strength"></span> Strength
                    </button>
                    <button class="filter-chip @(filterEndurance ? "filter-chip--active" : "")"
                            aria-pressed="@filterEndurance" @onclick="ToggleEndurance">
                        <span class="chip-dot chip-dot--endurance"></span> Endurance
                    </button>
                    <button class="filter-chip @(filterMixed ? "filter-chip--active" : "")"
                            aria-pressed="@filterMixed" @onclick="ToggleMixed">
                        <span class="chip-dot chip-dot--mixed"></span> Mixed
                    </button>
                </div>

                <div class="date-range">
                    <input type="date" class="date-input" @bind="dateStart" @bind:event="onchange"
                           @bind:after="OnDateRangeChanged" />
                    <span class="date-separator"> -- </span>
                    <input type="date" class="date-input" @bind="dateEnd" @bind:event="onchange"
                           @bind:after="OnDateRangeChanged" />
                </div>

                <select class="exercise-filter" @onchange="OnExerciseFilterChanged">
                    <option value="">All Exercises</option>
                    @foreach (var ex in loggedExercises)
                    {
                        <option value="@ex.Id">@ex.Name</option>
                    }
                </select>
            </div>
        </div>

        @{var filtered = GetFilteredSessions().ToList();}
        @if (filtered.Count == 0 && (searchText.Length > 0 || filterStrength || filterEndurance || filterMixed || dateStart.HasValue || dateEnd.HasValue || selectedExerciseId.HasValue))
        {
            <!-- Empty: no matching filters -->
            <div class="empty-state">
                <h2>No matching sessions</h2>
                <p>Try adjusting your filters or clearing the search.</p>
                <button class="btn-ghost" @onclick="ClearFilters">Clear Filters</button>
            </div>
        }
        else if (filtered.Count == 0)
        {
            <!-- Empty: no history at all -->
            <div class="empty-state">
                <h2>No workout history yet</h2>
                <p>Complete your first session to see it here. Start by scheduling a workout on the <a href="/calendar">Calendar</a>.</p>
            </div>
        }
        else
        {
            <!-- Session cards list -->
            <div class="session-list">
                @foreach (var (session, index) in filtered.Select((s, i) => (s, i)))
                {
                    <HistoryCard Session="session"
                                 IsExpanded="IsExpanded(session.WorkoutLogId)"
                                 OnToggle="() => ToggleCard(session.WorkoutLogId)"
                                 Index="index" />
                }
            </div>

            @if (GetRemainingCount() > 0)
            {
                <button class="btn-load-more" @onclick="LoadMore" disabled="@isLoadingMore">
                    @(isLoadingMore ? "Loading..." : $"Load More ({GetRemainingCount()} remaining)")
                </button>
            }
        }
    }
</div>
```
